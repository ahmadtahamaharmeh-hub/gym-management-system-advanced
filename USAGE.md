## نظام إدارة النادي الرياضي المتقدم

### الميزات الرئيسية 🎯

#### 1. **نظام الأعضاء والاشتراكات**
- إضافة وتعديل وحذف أعضاء
- إدارة خطط الاشتراك (شهري، ربع سنوي، نصف سنوي، سنوي)
- تتبع تاريخ انتهاء الاشتراكات
- تنبيهات تلقائية للاشتراكات المنتهية
- بحث متقدم وتصفية

#### 2. **نظام الحضور المتقدم**
- ربط مع أجهزة ZKTeco البيومترية
- تسجيل حضور الأعضاء والموظفين
- تقارير حضور يومية وأسبوعية
- معالجة الحالات الخاصة (تأخير، انصراف مبكر)
- مزامنة تلقائية مع جهاز البصمة

#### 3. **إدارة الموظفين والمدربين**
- تسجيل بيانات الموظفين والمدربين
- إدارة الأدوار والصلاحيات
- تتبع ساعات العمل
- إدارة ملفات المدربين المهنية

#### 4. **نظام التدريب المتقدم**
- إنشاء خطط تدريبية متنوعة
- رفع صور وفيديوهات توضيحية
- تصنيف الخطط حسب المستوى (مبتدئ، متوسط، متقدم)
- عرض الخطط للأعضاء
- معرض وسائط تفاعلي

#### 5. **إدارة الكافتيريا والمبيعات**
- إدارة منتجات الكافتيريا
- تسجيل المبيعات اليومية
- تتبع المخزون
- طرق دفع متعددة (نقد، بطاقة، محفظة)
- إيصالات بيع مفصلة

#### 6. **التقارير والإحصائيات**
- تقرير عام شامل
- تقارير الحضور
- تحليل المبيعات
- إحصائيات الاشتراكات
- الإيرادات والنفقات
- تقارير قابلة للتصدير

#### 7. **الأمان والتحكم بالوصول**
- نظام تسجيل دخول آمن
- أدوار وصلاحيات متعددة
- تشفير كلمات المرور
- حماية من هجمات XSS و SQL Injection
- سجلات تدقيق شاملة

## التثبيت 🚀

### الخطوة 1: استنساخ المستودع
```bash
git clone https://github.com/ahmadtahamaharmeh-hub/gym-management-system-advanced.git
cd gym-management-system-advanced
```

### الخطوة 2: إعداد قاعدة البيانات
```bash
mysql -u root -p
CREATE DATABASE gym_management;
USE gym_management;
SOURCE database/gym.sql;
```

### الخطوة 3: تكوين الإعدادات
عدّل ملف `config/config.php`:
```php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', 'your-password');
define('DB_NAME', 'gym_management');
define('ZKTECO_IP', '192.168.1.201');
```

### الخطوة 4: إعداد الأذونات
```bash
chmod -R 755 uploads
chmod -R 755 logs
```

### الخطوة 5: الوصول للنظام
```
http://localhost/gym-management-system-advanced/login.php
```

## ربط جهاز ZKTeco 🔗

### التثبيت:
```bash
composer require jmrashed/zkteco
```

### التكوين:
```php
// في config/config.php
define('ZKTECO_IP', '192.168.1.201');
define('ZKTECO_PORT', 23);
define('ZKTECO_TIMEOUT', 3000);
```

استخدم صفحة الحضور لمزامنة بيانات الحضور تلقائياً
